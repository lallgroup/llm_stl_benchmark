# To set this up
1. Clone the official repository `git clone git@github.com:hkust-nlp/Toolathlon.git`
2. Creat a virtual environment `python -m venv venv`
3. Activate the virtual environment `source venv/bin/activate`
4. Install the dependencies `pip install -r requirements.txt`
5. Run a basic test described in [EVAL_SERVICE_README.md](https://github.com/hkust-nlp/Toolathlon/blob/main/EVAL_SERVICE_README.md)
